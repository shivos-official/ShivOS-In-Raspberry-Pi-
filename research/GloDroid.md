# GloDroid Research

## Overview
GloDroid is an AOSP-based operating system focused on multiple ARM devices using clean Android architecture.

## Project Goal
Bring clean Android to ARM hardware.

## Architecture
- Pure AOSP
- Project Treble
- AIDL HAL
- Mainline Linux

## Graphics
- Mesa
- drm_hwcomposer
- minigbm

## Advantages
- Clean architecture
- Portable
- Upstream friendly
- Easy maintenance

## Disadvantages
- Hardware support may lag
- More difficult initial setup

## What ShivOS Can Learn
- Treble architecture
- HAL implementation
- Mainline kernel strategy
- Build system
- Device abstraction

## What ShivOS Should Avoid
- Ignoring Raspberry Pi specific optimizations
