# Android-x86 Research

## Overview
Android-x86 ports Android to desktop PCs.

## Architecture
- x86
- x64
- GRUB Bootloader

## Graphics
- Mesa
- drm_hwcomposer
- Vulkan
- OpenGL ES

## Desktop Features
- Window Mode
- Mouse Support
- Keyboard Shortcuts
- Desktop UI

## Advantages
- Mature desktop experience
- Strong graphics stack
- Excellent desktop support

## Disadvantages
- Not ARM native
- Different hardware ecosystem

## What ShivOS Can Learn
- Desktop mode
- Window manager
- Mouse interaction
- Keyboard shortcuts
- Desktop launcher
